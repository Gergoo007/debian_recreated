    About Debian Linux

    Debian is a Linux (GNU/Linux) distro that uses the dpkg package manager. The vast majority of the userland is part of the GNU project (coreutils, gcc, binutils, bash). It uses the systemd init system.

    License

    Debian uses the DFSG license.

    The APT package manager

    About APT

    APT is a frontend to the dpkg package manager, designed to make using dpkg easier. It's license is GPLv2.
    It works by downloading a database containing package names, their download url and their version.

    Using APT

    First, you have to update this database. That can be done by issuing

    apt update

    Don't forget to run it as root!
    Then run

    apt upgrade

    To download and install the packages from the fetched database.
