    How to flash the installer ISO

    Flash the ISO to either a CD or pendrive, preferably using Rufus, or using dd:

    dd if=iso_image of=device bs=4M

    Then boot the pendrive/CD you just flashed.

    Using the installer

    - After the ISO is booted, select "Graphical Install" in the GRUB menu.

    - Select your language, region and keymap

    - Wait a bit then enter your new hostname and domain when prompted.

    - Enter your root password (it's important to remember it).

    - Enter a name and a password for your user account.

    - When partitioning, select "Use entire disk" and separate /home partition

    - Use your common sense
